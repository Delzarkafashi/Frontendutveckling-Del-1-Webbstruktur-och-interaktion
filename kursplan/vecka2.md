# Vecka 2 – HTML & CSS – Grunder

## Veckans syfte

Syftet med vecka 2 är att lära sig grunderna i HTML och CSS genom att bygga en enkel webbsida med tydlig struktur och grundläggande styling.

---

## Vad vi ska göra

Under vecka 2 ska vi arbeta med följande moment:

- Skriva HTML med semantiska element (header, main, footer, section, nav)
- Använda listor, länkar och bilder på rätt sätt
- Skapa enkla formulärelement (t.ex. input och button)
- Styla med CSS: färger, text och box-model (margin, padding, border)
- Bygga layout med Flexbox och/eller Grid
- Göra en enkel responsiv anpassning (så sidan funkar på mindre skärmar)
- Mini-projekt: profilsida eller enkel layout (t.ex. startsida)

---

## Hur ska vi tänka

Under vecka 2 ska vi fokusera på att få struktur och layout att sitta.

Vi ska:
- Börja med HTML-strukturen innan vi stylar
- Använda semantiska taggar för att göra sidan tydlig och lätt att förstå
- Styla steg för steg (testa ofta och ändra lite i taget)
- Tänka “enkelt först” och förbättra efteråt

Målet är att kunna bygga en hel sida med HTML + CSS som ser bra ut och har en tydlig struktur.

---

## Pseudokod

1. Bygg HTML-struktur
   - Skapa header, main och footer
   - Lägg in nav och grundinnehåll

2. Lägg till innehåll
   - Lägg in länkar, listor och bilder
   - Lägg in formulärelement (input + knapp)

3. Styla grunder
   - Sätt färger, font och spacing
   - Använd box-model (margin/padding/border)

4. Layout
   - Använd Flexbox/Grid för att placera innehåll
   - Justera för mobil (enkel responsiv design)

5. Testa och förbättra
   - Kontrollera att allt ser bra ut i webbläsaren
   - Fixa smådetaljer och städa CSS
