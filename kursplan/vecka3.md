# Vecka 3 – JavaScript: Grunder

## Veckans syfte

Syftet med vecka 3 är att bygga en stabil grund i **JavaScript-logik**.
Vi tränar på att tänka i kod: spara data, fatta beslut, upprepa saker och lösa
enkla problem.

---

## Vad vi ska göra

Under vecka 3 ska vi arbeta med följande moment:

- Variabler (`let`, `const`)
- Datatyper (string, number, boolean)
- Villkor (`if / else`)
- Loopar (`for`, `for...of`)
- Funktioner (skapa och anropa)
- Arrays och objekt (grundnivå)

**Mini-projekt / övningar:**
- Miniräknare (enkel logik)
- Logikuppgifter (t.ex. jämförelser och loopar)

---

## Hur ska vi tänka

Under vecka 3 fokuserar vi på **logik före UI**.

Vi ska:
- Börja enkelt och bygga steg för steg
- Använda `console.log` för att förstå vad som händer
- Skriva tydliga variabel- och funktionsnamn
- Tänka i flödet: input → logik → output
- Träna på att felsöka istället för att stressa

Målet är inte att skriva mycket kod, utan att **förstå hur JavaScript fungerar**.

---

## Pseudokod

1. Skapa data
   - Skapa variabler med `let` och `const`
   - Testa olika datatyper (text, tal, true/false)

2. Villkor
   - Skriv `if / else`
   - Exempel: om en produkt finns i lager → skriv ut ett meddelande

3. Loopar
   - Loopa igenom en array
   - Exempel: skriv ut alla storlekar eller färger

4. Funktioner
   - Skapa en funktion som gör en sak
   - Exempel: räkna pris, kontrollera lagerstatus
   - Anropa funktionen och skriv ut resultatet

5. Arrays och objekt
   - Skapa ett objekt med flera properties
   - Använd arrays inuti objektet
   - Läs ut data med `object.property`

6. Testa och felsök
   - Använd `console.log`
   - Kontrollera stavning, ordning och värden
